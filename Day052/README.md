  - `DCPS3`: Debug switch to Exception level 3

  - `MRS`: Move System register to general-purpose register
  - `MSR`: Move general-purpose register to System register

  - `MRRS`: Move 128-bit System register to two adjacent 64-bit general-purpose registers
  - `MSRR`: Move two adjacent 64-bit general-purpose registers to 128-bit System register
