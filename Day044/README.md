# Day 44

Notes reading [Doc 1](../Doc001.md).

* p.323
  - Device memory attribute:
    - `G`: location has the Gathering attribute
    - `nG`: location does not have the Gathering attribute, meaning it is non-Gathering
  - Note: Gathering can be performed if a programmer cannot tell whether gathering has occurred

* p.324
  - Device memory attribute:
    - `R`: location has the Reordering attribute
    - `nR`: location does not have the Reordering attribute

* p.325
  - Device memory attribute:
    - `E`: location has the Early Write Acknowledgement attribute
    - `nE`: location has the No Early Write Acknowledgement attribute
